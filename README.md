# AI-Based QA System

An AI-powered Question Answering System built using Python, Flask, FAISS, Retrieval-Augmented Generation (RAG), OCR, and Large Language Models.

## Features

* Document Upload and Processing
* PDF Text Extraction
* OCR Support for Scanned Documents
* Vector Database using FAISS
* Retrieval-Augmented Generation (RAG)
* Semantic Search
* Query Rewriting
* Semantic Caching
* Real-Time Streaming Responses
* User Authentication
* SQLite Database Integration

## Tech Stack

### Backend

* Python
* Flask

### AI & Machine Learning

* LangChain
* FAISS
* Sentence Transformers
* OpenAI / OpenRouter APIs

### Database

* SQLite

### Frontend

* HTML
* CSS
* JavaScript

### Deployment

* Docker
* Kubernetes

## Installation

```bash
git clone https://github.com/KPatil2003/AI--Based-QA--System.git
cd AI--Based-QA--System
pip install -r requirements.txt
python src/app.py
```
